### Install

### Basic Setup
```javascript
import { UmiAgentKit } from 'umi-agent-kit';

// Initialize with AI and Multisig
const kit = new UmiAgentKit({
  network: 'devnet',
  multisigEnabled: true
});

  groqApiKey: 'your-groq-api-key'
});

// Import your wallet
```
